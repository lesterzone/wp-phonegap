## Phonegap Based Project

## Hard dependencies

phonegap

```
npm i phonegap -g
```

## Local dependencies

```
npm i
```

## Builds

android

```
npm run build-android
```

ios

```
npm run build-android
```

## About phonegap packages

Follow instructions for each platform and how to install from there
